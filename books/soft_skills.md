
### Chapter.71 「失敗を恐れるな」

なぜ人々は失敗を恐れるのか。

→ 壊れやすい自我を守ることが何よりも大事だから。
→ 失敗を個人的な体面の問題としてと捉えすぎてしまい、特定の分野での失敗は、自分個人としての価値を傷つけると思っているため。

```
失敗に対する恐怖を拭い去るための第一歩は、自分で失敗したまま終わりにしない限り、失敗で終わりにならないことを理解することだ。
```

世間がイメージする失敗は存在しない。あくまで成功までのマイルストーンに過ぎない。

「勝つか学ぶか」。

失敗を喜び、期待し、受け入れ、失敗に正面から向き合うことを学ぼう。

```
もっとも偉大な人々は、最大の失敗を乗り越えてたった一歩進んだときに最大の成功を獲得している。
```
